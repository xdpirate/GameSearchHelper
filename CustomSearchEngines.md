# Custom search engines
## Introduction
This page provides some pre-made custom search engines you can use with Game Search Helper, listed in alphabetical order. This list may grow substantially in the future, so use your browser's search function to find the one you want (Ctrl+F / Cmd+F).

Icons are included. After installation, you can edit the search engine in the GSH options like any other custom search engine.

### **This page is dynamic!** 
To use it, first [install Game Search Helper](https://github.com/xdpirate/GameSearchHelper), then visit/reload this page and click the installation link next to the custom search engine you would like to add to GSH.

If GSH is not detected, try reloading the page. If you are running Tampermonkey on Firefox, you might need to disable and reenable TM from the extensions page to make it work.

Feel free to pull request to add new search engines to this page!
